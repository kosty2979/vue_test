# Vue 3 webpack boilerplate



## Includes

- Vue 3.0.0-beta.18
- Vue Router 4.0.0-beta.1
- Vuex 4.0.0-beta.4
- Babel 7.10.4
- Webpack 4.43.0


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
